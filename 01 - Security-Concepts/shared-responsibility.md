
# Shared Responsibility Model

## Overview

The Shared Responsibility Model defines the security responsibilities shared between the cloud provider and the customer.

## Microsoft's Responsibilities

Microsoft is responsible for:

- Physical datacenters
- Physical hosts
- Network infrastructure
- Azure platform services

## Customer Responsibilities

Customers are responsible for:

- Identities
- Devices
- Data
- Access management
- Endpoint protection
- Application configuration

## Why It Matters

Moving to the cloud does not transfer all security responsibilities to the cloud provider.

Organizations remain responsible for protecting their users, identities, and data.

## Key Takeaways

- Security is a shared effort.
- Customer responsibility depends on the service model.
- Identities and data are always critical responsibilities.
